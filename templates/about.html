About econgraphs.org

My name is Chris Makler. I've taught econ classes at the University of Pennsylvania and Stanford University, and worked as the Senior Economist for Aplia, back when all Aplia did was econ. :)

I've created econgraphs.org as an open source project to provide econ teachers with a standard set of interactive graphs they can use in classes.

Most of the graphs here are actually sets of linked graphs -- for example, individual and market demand, or total and marginal costs for a firm -- so that econ teachers can illustrate the important underlying relationships within an economic model.

Comments and suggestions are very much appreciated; send me a note at chris dot makler at gmail dot com. If you're technically minded, check out the source code on Github -- and if you're feeling really adventurous, add to the code and send me a pull request!

Acknowledgements

I am trained as an economist, not as a software engineer; and this project would not have been possible without the mentorship of an outstanding group of colleagues over the years, including (but certainly not limited to!) David Murphy, Karl Lew, Wilson Cheung, Harrison Caudill, and Kyle Moore.

This project was made far easier by the amazing Javascript libraries currently available. In particular, I've benefited greatly from D3.js to draw the graphs, angular.js to make them dynamic, katex to add beautiful mathematical labels to them, and reveal.js to incorporate them into presentation slides.
