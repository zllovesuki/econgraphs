Group Work: Utility Functions

For each of the following function types, have each member of your team find the MU_x, MU_y, and MRS_{x,y} for one. Compare your answers and then answer the following questions:

  1. Do the first five utility functions have the same MU_x and MU_y?
  2. Do the first five utility functions have the same MRS_{x,y}?
  3. Confirm that the first five utility functions represent the same set of preferences by choosing two random (strictly positive) bundles (x_1,y_1) and (x_2,y_2) and confirming that all five rank those bundles the same way.
  4. Find the value of \alpha such that the sixth utility function represents the same set of preferences as the first five utility functions.
  5. Draw the indifference curve for this utility function [i.e., the sixth function, with the value of \alpha you found in part (d)] passing through the point (27,27). Draw the tangent line to the indifference curve at that point, and indicate its slope.
  6. In each type of utility function, there is a sense in which the consumer likes good Y "twice as much" as good X. Can you find a way to verbalize what that means for each of these types of utility functions?

1. Cobb Douglas Utility Functions

u(x,y) = xy^2 u(x,y) = 100xy^2 u(x,y) = \frac{1}{2}\ln x + \ln y
u(x,y) = x^{1/10}y^{1/5} u(x,y) = \ln x + 2\ln y u(x,y) = x^\alpha y^{1-\alpha}

2. Perfect Substitute Utility Functions

u(x,y) = x + 2y u(x,y) = 100x + 200y u(x,y) = \frac{1}{2}x + y
u(x,y) = \frac{1}{10}x + \frac{1}{5}y u(x,y) = 8x + 16y u(x,y) = \alpha x + (1-\alpha) y

3. Perfect Complement Utility Functions

u(x,y) = \min\{2x, y\} u(x,y) = \min\{200x, 100y\} u(x,y) = \min\{x, \frac{1}{2}y\}
u(x,y) = \min\{\frac{x}{5},\frac{y}{10}\} u(x,y) = \min\{\frac{x}{8},\frac{y}{16}\} u(x,y) = \min\{\frac{x}{\alpha},\frac{y}{1-\alpha}\}
