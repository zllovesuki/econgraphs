Group Work: Deriving Supply for Cobb-Douglas Production Function

Once again consider a firm with production technology f(L,K)=L^\frac{1}{4}K^\frac{1}{4}. In the short run, capital is fixed at \overline{K} = 4); in the long run capital and labor can be freely adjusted.

We've shown before that the short-run and long-run total costs for this production function are:

TC(q) = 2\sqrt{wr}q^2

STC(q) = r\overline{K} + wL(q) = 4r + \frac{w}{4}q^4

  1. Derive firm's marginal cost function in the short run and long run.
  2. Derive the firm's supply function, q(P), in the short run and the long run. Plot the long-run supply and short-run supply on the same graph.
  3. Derive expressions for the firm's profit in the short run and the long run, given the optimal supply decision.
  4. Find the firm's profit-maximizing demand for labor, L^*(w,r,P), as well as its conditional demand for labor, L^*(w,r,q). Plot these on two graphs, one for the long run and one for the short run.
