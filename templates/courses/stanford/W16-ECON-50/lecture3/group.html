Group Work: Elasticity

Cheat sheet: If Y is a function of X, then the "X elasticity of Y" \left(\epsilon_{Y,X}\right) may be calculated using one of the following three formulas:

midpoint method

\epsilon_{Y,X} = \frac{\Delta Y/Y_M}{\Delta X/X_M}

where \Delta X and \Delta Y are the difference
between two X and Y values, and X_M and Y_M
are the midpoints between those values

point method

\epsilon_{Y,X} = \frac{dY/Y}{dX/X} = \frac{dY}{dX} \times \frac{X}{Y}

log-log method

\epsilon_{Y,X} = \frac{d \ln Y}{d \ln X}

  1. Calculating Elasticity for a Specific Linear Demand Curve

    Use the point method to find the price elasticity of demand if Q^D(P) = 12 - 2P.

    1. When is \epsilon perfectly inelastic?
    2. When is \epsilon perfectly elastic?
    3. When is \epsilon unit elastic?

  2. Calculating Elasticity for a General Linear Demand Curve

    Repeat question 1 for a general linear demand function Q^D(P) = a - bP.

  3. Calculating Elasticity for a Constant Elasticity Supply Curve

    Consider the supply function Q^S(P) = P^2.

    1. Use the midpoint method to calculate the price elasticity of supply between P = 99 and P = 101.
    2. Use the point method to calculate the price elasticity of supply at P = 100.
    3. Use the log-log method to calculate the price elasticity of supply at P = 100.

  4. Deriving the linear relationship between log data

    Yule for You, Inc. has a popular brand of “Natural Yule Logs” which burn with 50% less smoke than real wood. Ella Sticetti, the owner of Yule for You, decided to experiment a bit with pricing on her web site, randomly choosing different prices for different customers to see how responsive demand was to changes in price. The data she found is shown in the table below.

    Price (P) Quantity Demanded (Q^D) \ln P \ln Q^D
    $2.56 50 0.94 3.91
    $4.00 40
    $6.25 32
    $10.24 25
    $16.00 20
    1. Using this data, sketch this demand curve in a diagram with Q^D on the horizontal axis and P on the vertical axis.
    2. Now fill out the rest of the table by calculating \ln P and \ln Q^D for each line of the table, using a calculator or spreadsheet. (The first line has been done for you.) Plot these points on a new graph with \ln Q^D on the horizontal axis and \ln P on the vertical axis.
    3. All of the dots should appear in a line; find the slope, and use the point-slope formula to calculate the equation of that line and use the result to write \ln Q^D as a function of \ln P. (Be sure to remember that the dependent variable is on the horizontal axis!)
    4. Write down the demand curve that Yule for You faces for their Natural Yule Logs, Q^D(P). (Hint: what's the inverse function of a natural log...?)
    5. What is Yule for You's price elasticity of demand for Natural Yule Logs at P = \$4.00? What about at P = \$6.25?
