Group Work: Deriving the Slutsky Diagram for Cobb-Douglas

Consider the "canonical" Cobb-Douglas utility function u(x,y) = xy. Remember that the (Marshallian) demand functions for this utility function are x^*(P_x,P_y,I) = \frac{I}{2P_x} and y^*(P_x,P_y,I) = \frac{I}{2P_y}.

  1. Derive the indirect utility function

    Plug the optimal bundle (x^*(P_x,P_y,I),y^*(P_x,P_y,I)) into the utility function to find the indirect utility function V(P_x,P_y,I) = u(x^*(P_x,P_y,I),y^*(P_x,P_y,I))

  2. Derive the expenditure function

    Find the necessary income required to achieve utility U by setting V(P_x,P_y,I) = U and solving for I. Call this the "expenditure function" E(P_x,P_y,U).

  3. Derive the Hicksian demand

    Substitute this income into the Marshallian demand functions to find the Hicksian demand functions: x^*(P_x,P_y,E(P_x,P_y,U)) and y^*(P_x,P_y,E(P_x,P_y,U)).

  4. Create the Slutsky diagram

    In this part we'll use what you derived above to create the Slutsky decomposition diagram for the price change from P_{x_1} = 1 to P_{x_2} = 4, holding P_y = 1 and I = 8.

    • Draw the budget lines for (P_x = 1, P_y = 1,I = 8) and (P_x = 4, P_y = 1,I = 8).
    • Find the optimal consumption bundle (Marshallian demand) for those two budget constraints; add them to your diagram as points A and C.
    • Plug (P_x = 1, P_y = 1,I = 8) into the indirect utility function (or just evaluate the utility at bundle A!) to find the utility at the initial prices and income, U_1 = V(P_x,P_y,I) = V(1,1,8).
    • Plug U_1 into the expenditure function to determine the income required to afford this initial utility at the new price; i.e., E_D = E(P_x, P_y, U) = E(4, 1, U_1); add the "decomposition budget line" 4x + y = E(4,1,U_1) to your diagram.
    • Plug U_1 into the Hicksian demand functions determine the decomposition bundle; add this point to your diagram as point B.
    • Add the indifference curve for U_1 to your diagram.
