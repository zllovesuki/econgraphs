But we're saved...by indifference curves!

It turns out that we can get
everything we need to derive and analyze demand
from indifference curves -- and in particular, from
the Marginal Rate of Substitution.

So as long as a utility function
generates the right indifference curves,
the "utility numbers" don't actually matter!

Recall: Fundamental Assumptions

Preferences are complete. Any two bundles can be compared. For any A and B,
either A \succeq B or B \succeq A.
Preferences are transitive. If you like A at least as much as B,
and you like B at least as much as C,
then you like A at least as much as C. 		If A \succeq B and B \succeq C,
then A \succeq C.

We can achieve this by using a utility function u()
to assign a numerical value to each bundle:

A \succ B if and only if u(A) > u(B).

A \sim B if and only if u(A) = u(B).

A \prec B if and only if u(A) < u(B).

Recall: Monotonicity Assumption

Bundle A is preferred to bundle B (that is, A \succ B)
if A contains at least as much of all goods as B,
and strictly more of at least one good.

In other words, holding all other quantities in a bundle constant,
an increase in the quantity of one good results in an increase in utility.

Now, with our multivariate utility function u(x,y),
we see that this is an assumption about the partial derivatives of that function:

\displaystyle{\frac{\partial u(x,y)}{\partial x} > 0}

\displaystyle{\frac{\partial u(x,y)}{\partial y} > 0}

Marginal utility of good X (MU_x)
is positive.

Marginal utility of good Y (MU_y)
is positive.

Recall: Indifference Curves

The indifference curve for bundle A shows all the points the consumer likes as much as A
(is "indifferent" between that bundle and A).

Now: set of all bundles that give the same utility as bundle A
...that is, the set of all (x,y) such that u(x,y) = u(x_A,y_A)

Recall: Marginal Rate of Substitution (MRS)

The rate at which the consumer is just willing to substitute one good for another

Visually: The negative of the slope of the indifference curve

Mathematically: The ratio of the marginal utilities, \frac{MU_x}{MU_y}

Marginal Rate of Substitution (MRS)

Moving from point A to point B

Deriving the MRS from the utility function

{% raw %}

Consider the point C = (x_A,y_B).

U(A) - U(C) = U(x_A, y_A) - U(x_A, y_B)
\displaystyle{= (y_A - y_B) \times \frac{U(x_A, y_A) - U(x_A, y_B)}{y_A - y_B}}
\displaystyle{\rightarrow (y_A - y_B) \times \frac{\partial u(x,y)}{\partial y}} as B \rightarrow A
U(B) - U(C) = U(x_B, y_B) - U(x_A, y_B)
\displaystyle{= (x_B - x_A) \times \frac{U(x_B, y_B) - U(x_A, y_B)}{x_B - x_A}}
\displaystyle{\rightarrow (x_B - x_A) \times \frac{\partial u(x,y)}{\partial x}} as B \rightarrow A

These two must be equal!!!

 {% endraw %}

Deriving the MRS from the utility function

{% raw %}

(Continuing) C = (x_A,y_B). As B \rightarrow A:

U(A) - U(C) \displaystyle{\rightarrow (y_A - y_B) \times \frac{\partial u(x,y)}{\partial y}}
U(B) - U(C) \displaystyle{\rightarrow (x_B - x_A) \times \frac{\partial u(x,y)}{\partial x}}

These two must be equal!!!

U(A) - U(C) = U(B) - U(C)
\displaystyle{(y_A - y_B) \times \frac{\partial u(x,y)}{\partial y}} = \displaystyle{(x_B - x_A) \times \frac{\partial u(x,y)}{\partial x}}
\displaystyle{\frac{y_A - y_B}{x_B - x_A}} = \displaystyle{ \frac{\partial u(x,y) / \partial x} {\partial u(x,y) / \partial y} }
\displaystyle{MRS = -\frac{y_B - y_A}{x_B - x_A}} = \displaystyle{\frac{MU_x}{MU_y}}
{% endraw %}

A simpler, equivalent but less precise derivation
(somewhat following B&B, page 91):

Suppose a consumer changes the level of consumption of x and y by \Delta x and \Delta y.

The corresponding impact on utility will be:

\Delta U = MU_x \times \Delta x + MU_y \times \Delta y

But if we're along an indifference curve, \Delta U = 0; so it must be that:

MU_x \times \Delta x + MU_y \times \Delta y = 0

Subtract MU_x \times \Delta x from both sides:

MU_y \times \Delta y = -MU_x \times \Delta x

Multiply both sides by MU_y and divide both sides by -\Delta x:

\displaystyle{-\frac{\Delta y}{\Delta x} = \frac{MU_x}{MU_y}}

FUNDAMENTAL FORMULA FOR ECON 50

MRS_{x,y} = \frac{MU_x}{MU_y}

...and note the units!

\displaystyle{\frac{MU_x}{MU_y} = \frac { \frac{ \text{utils} }{ \text{units of } X } } { \frac{ \text{utils} }{ \text{units of } Y } }= \frac{\text{units of }Y}{\text{units of }X}}

The utils cancel, leaving us with only units of X and Y.

Utility Function I:
Cobb-Douglas

\displaystyle{u(x,y) = x^ay^b}

Generic Cobb-Douglas:

\displaystyle{u(x,y) = cx^ay^b}

MU_x =

acx^{a-1}y^b

MU_y =

bcx^ay^{b-1}

MRS_{x,y} =

\displaystyle{\frac{MU_x}{MU_y} =}

\displaystyle{\frac{acx^{a-1}y^b}{bcx^ay^{b-1}} =}

\displaystyle{\frac{ay}{bx}}

We've just shown that if \displaystyle{u(x,y) = cx^ay^b}, then \displaystyle{MRS_{x,y} = \frac{ay}{bx}}.

Find the marginal rate of substitution
for the following Cobb-Douglas utility functions:

u(x,y) = xy^2

\Rightarrow a = 1, b = 2

\Rightarrow MRS = \frac{1y}{2x} = \frac{y}{2x}

u(x,y) = x^{1/3}y^{2/3}

\Rightarrow a = \frac{1}{3}, b = \frac{2}{3}

\Rightarrow MRS = \frac{\frac{1}{3}y}{\frac{2}{3}x} = \frac{y}{2x}

u(x,y) = x^{1/10}y^{1/5}

\Rightarrow a = \frac{1}{10}, b = \frac{1}{5}

\Rightarrow MRS = \frac{\frac{1}{10}y}{\frac{1}{5}x} = \frac{y}{2x}

u(x,y) = x^{8}y^{16}

\Rightarrow a = 8, b = 16

\Rightarrow MRS = \frac{8y}{16x} = \frac{y}{2x}

u(x,y) = 4xy^2

\Rightarrow a = 1, b = 2

\Rightarrow MRS = \frac{1y}{2x} = \frac{y}{2x}

Utility Function II: Perfect Substitutes

Generic Perfect Substitutes:
\displaystyle{u(x,y) = c + ax + by}

MU_x =

a

MU_y =

b

MRS_{x,y} =

\displaystyle{\frac{MU_x}{MU_y} =}

\displaystyle{\frac{a}{b}}

We've just shown that if \displaystyle{u(x,y) = ax + by + c}, then \displaystyle{MRS_{x,y} = \frac{a}{b}}.

Find the marginal rate of substitution
for the following perfect-substitute utility functions:

u(x,y) = x + 2y

\Rightarrow a = 1, b = 2

\Rightarrow MRS = \frac{1}{2}

u(x,y) = \frac{1}{3}x + \frac{2}{3}y

\Rightarrow a = \frac{1}{3}, b = \frac{2}{3}

\Rightarrow MRS = \frac{\frac{1}{3}}{\frac{2}{3}} = \frac{1}{2}

u(x,y) = \frac{1}{10}x + \frac{1}{5}y

\Rightarrow a = \frac{1}{10}, b = \frac{1}{5}

\Rightarrow MRS = \frac{\frac{1}{10}}{\frac{1}{5}} = \frac{1}{2}

u(x,y) = 8x + 16y

\Rightarrow a = 8, b = 16

\Rightarrow MRS = \frac{8}{16} = \frac{1}{2}

u(x,y) = x + 2y + 4

\Rightarrow a = 1, b = 2

\Rightarrow MRS = \frac{1}{2}

Generic Perfect Complements:
\displaystyle{u(x,y) = \min(ax,by)}

MU_x =

a \text{ if } ax < by \text{; otherwise 0}

b \text{ if } by < ax \text{; otherwise 0}

Summary

Utility functions represent preferences by assigning a numerical value ("utility") to each bundle.

Any utility functions that give us the same indifference curves, will give us the same MRS.

The MRS is equal to the ratio of the marginal utilities.

Three utility functions we will be coming back to over and over again are Cobb-Douglas, perfect complements, and perfect substitutes.

In fact, these are three special cases of constant elasticity of substitution functions.
