(e.g. the number of gallons of gasoline purchased by a particular consumer)

Summary

  • ELASTICITY IS NOT SLOPE
  • Be able to conceptualize elasticity in several ways:
    • perfectly elastic vs. perfectly inelastic
    • more elastic vs. less inelastic
    • graphically (the shape of a curve) vs. numerically (a precise number)
  • Be able to calculate elasticity for different functional forms:
    • linear (e.g., )
    • multiplicative (e.g., ) - using natural log trick
  • Note that all elasticities are defined in similar ways (% change / % change)...but don't confuse them with each other!
    • For example, income and price elasticities give us different information
  • Elasticity is usually not constant - so it may be quite different in different portions of the same demand (or supply) curve. Also, ELASTICITY IS NOT SLOPE.
