Marginal Returns and Returns to Scale

A firm exhibits diminishing marginal product of labor if:

\displaystyle{\frac{\partial ^2f(L,K)}{\partial L^2}} < 0

A firm exhibits decreasing returns to scale if:

f(tL,tK) < tf(L,K)

for some t > 1

For Cobb-Douglas of the form f(L,K) = L^\alpha K^\beta:

\displaystyle{\frac{\partial f(L,K)}{\partial L} = \alpha L^{\alpha-1} K^\beta}

\displaystyle{\frac{\partial ^2f(L,K)}{\partial L^2} = \alpha(\alpha - 1)L^{\alpha-2}K^\beta}

Profit-Maximizing Demand for Labor

Step 1: Find conditional demand for labor, L^*(w,r,q)

Step 2: Find optimal quantity for price P, q*(P)

Step 3: Plug q*(P) into L^*(q) to get L^*(w,r,q^*(P))

Note

Short-run conditional demand for labor was perfectly inelastic.

Short-run profit-maximizing demand for labor is not! ...but why not?

Why is the short-run profit-maximizing demand for labor not perfectly inelastic?

The firm is adjusting its capital The firm has some pricing power The firm is adjusting quantity in response to prices The firm has some wage-setting power

Conditional on q, a change in w
will not affect the amount of labor used

...but a change in w will cause
a change in the profit-maximizing q

...so the labor needed to produce the profit-maximizing q
will depend on w

Hotelling's Lemma

\displaystyle{\frac{\partial \pi(w,r,P)}{\partial P} = q^*(w,r,P)}

...in the short run or the long run

Why? ...a little like Shephard's Lemma

\pi(P)

=P \times q^*(P) - TC(q^*(P))

\displaystyle{\frac{\partial \pi(P)}{\partial P}}

\displaystyle{=q^*(P) + P\frac{\partial q^*}{\partial P} - \frac{\partial TC}{\partial q}\frac{\partial q^*}{\partial P}}

\displaystyle{=q^*(P) + [P - MC(q)]\frac{\partial q^*}{\partial P}}

...and since P = MC(q), the second term is zero.

The Entry/Exit Decision

Even in the long run, there is another "economic" cost:

the opportunity cost of the firm's next best option.

Let's call this \pi_{\text{other}}

A firm will exit the industry if:

Profit if Stay

<

Profit if Leave

\pi(\color{red}{q})

<

\pi_\text{other}

TR(\color{red}{q}) - TC( \color{red}{q})

<

\pi_\text{other}

TR(\color{red}{q})

<

\pi_\text{other} + TC( \color{red}{q})

(sort of like FC + TVC( \color{red}{q}))

A firm's very-long run cost function includes this opportunity cost!

This results in a U-shaped average cost even if all inputs are flexible.

SUPPLY DECISIONS: SUMMARY

Short run supply: adjust variable inputs (e.g. L)

to set q so that P = MC(q),

as long as P > \min AVC(q)

Long run supply: adjust all inputs (e.g., K and L)

to set q so that P = MC(q),

as long as P > \min AC(q),

including opportunity costs of leaving industry.
